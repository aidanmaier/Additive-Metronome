# Additive Metronome

A simple metronome for additive rhythms: easily construct irregular meters or groupings from 2 and 3-beat durations.

### Features
* Tap: to set tempo
* Flash: visual metronome
* Fill: play all sub-divisions

### Development
Event-driven single-page app built with:
* TypeScript
* React
* Vite
* Material UI [https://github.com/mui/material-ui]
